<h1>Controls</h1>

<h2>Player One</h2>

Move left : Q
Move right : D
Move down faster : W

Rotate clockwise : Space
Rotate anti-clockwise : LeftAlt

<h2>Player Two</h2>

Move left : Left Arrow
Move right : Right Arrow
Move down faster : Down Arrow

Rotate clockwise : Right Ctrl
Rotate anti-clockwise : Alt Gr
